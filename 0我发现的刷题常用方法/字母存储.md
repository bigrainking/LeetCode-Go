

### **1. 情况**：

对于全是字母的情况的字符串，如果需要字母相关信息，用[26]Type{}数组存储对应信息



**为甚使用26？**

小写字母的变化范围是26 ~ 26*2， 每个字母-26就可以减小数组的长度



### **2. 举例：**

计算一个全是小写字母的字符串的每个字母出现次数

```go
count := [26]int{}
for _, s := range str {
   count[s-'a']++ //用s-‘a’是为了保证数组长度最大26，否则就是52
}
```
