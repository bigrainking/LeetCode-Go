# 二、解析

- 使用插入哈希集的方法

   还有别的方法没有做

```go
func singleNumber(nums []int) int {
    // 第一次出现则标记为false，第二次出现标记为true
    // 返回记录为false的元素key值
    // 时间复杂度：O(N)
    // 空间复杂度：O(N)
    hashSet := map[int]bool{}
    for _, num := range nums {
        if _, ok := hashSet[num]; ok {
            hashSet[num] = true //重复元素true
        }else {
            hashSet[num] = false // 单一元素false
        }
    }

    for key, value := range hashSet {
        if value == false {
            return key
        }
    }
    return -1
}
```
